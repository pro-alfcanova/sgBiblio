Olá pessoal,

Antes de qualquer coisa aviso que não estou programador, eu sei programar um pouco e todos sabemos que são coisas muito diferentes!

Eu estou professor da rede pública do Estado de São Paulo e as escolas possuem bibliotecas ou salas de leitura, que foram o foco desse nosso projeto.

Resolvemos implatar um sistema de gerenciamento de bibliotecas/salas de leitura de baixo custo, ou seja, que não precisassem de nenhum outro recurso,
tais como: computadores dedicados, servidores alugados entre outros.

Desse modo, resolvemos utilizar a infraestura do Google Apps Script (GAS), que possue certas limitações que nos conduziram até esse código.

Sei que deveriamos revisar e refatorar páginas HTML, estilos CSS e os códigos JavaScrips, e talvez nós façamos em versões futuras.

Sendo muito otimista, nosso DB são planilhas do GoogleSheets, uma para cada função: leitores, livros, empréstimos e usuários.

Anexamos algumas planilhas no formato XLSX como modelo e estão pré preenchidas com casos teste.

Deixamos aqui para vocês os códigos e se puderem contribuir agradeçemos imensamente.
